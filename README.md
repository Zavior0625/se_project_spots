# Spots

## 📖 Project Description

**Spots** is a responsive web application that showcases a user profile and a collection of cards, built as part of the **TripleTen** front-end course.  

The project includes:

- A **profile section** with avatar, name, description, edit profile, and new post buttons.
- A **cards section** displaying images with titles and like buttons.
- Responsive layout for **desktop, tablet, and mobile**.
- Hover effects and text overflow handling for clean, interactive UI.

---

## 🛠️ Technologies and Techniques

- **HTML5** for semantic structure
- **CSS3** for styling, including:
  - Flexbox for profile layout
  - CSS Grid with `auto-fit` and `minmax()` for card layout
  - Hover states for buttons
  - Ellipsis handling for single-line and multi-line text overflow
- **Fonts** via `@font-face` (Poppins)
- **Normalize.css** for cross-browser consistency
- **Responsive design techniques** (media queries, percentage-based widths, max-width constraints)
